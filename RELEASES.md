# Obscura 版本更新跟踪

## v0.1.9 (当前版本)

**发布日期**：2026-06-28  
**Release 页面**：https://github.com/h4ckf0r0day/obscura/releases/tag/v0.1.9  
**提交数**：59 commits since v0.1.8

### 亮点
- Request/response interception API（`on_request`/`on_response`）
- `--stealth` 变为全局标志，覆盖所有子命令
- stealth 大重构：creepjs 报告 0% 检测率
- 新增 `--dump cookies`、`Network.setBlockedURLs`、`Network.getResponseBody`
- 新增 `document.evaluate` / `XPathResult`（XPath 子集）
- 大量 CDP、Web API 和构建修复
